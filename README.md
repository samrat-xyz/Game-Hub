Project Name: Gamehub - A Game Library
Project Description:

This project is an online game library where users can explore various indie games, view detailed information about each game, and install the ones they like. It is an interactive and responsive web application built using React and Firebase.

Features:

Users can browse different indie games easily.

Each game has a detailed information page (name, developer, genre, and description).

Firebase Authentication is used for user login and registration.

Logged-in users can view their profile information.

The design is attractive and responsive using DaisyUI.

A dynamic banner slider is added using Swiper.js.

React Context API is used for global state management.

JSON data is used to store and display game information.

Technologies Used:

React.js – For frontend development

DaisyUI – For beautiful and responsive UI design

React Router – For page navigation and routing

Swiper.js – For creating sliders and banners

Firebase – For authentication and hosting

React Context API – For managing authentication state and global data

JSON Data – For storing and displaying game data

React Icons – For using attractive icons
